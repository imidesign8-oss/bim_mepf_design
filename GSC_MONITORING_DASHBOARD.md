## Weekly Monitoring Checklist

### Monday Morning Check-In

**Tasks (15 minutes):**
- [ ] Check GSC dashboard for any critical errors
- [ ] Review total impressions (compare to previous Monday)
- [ ] Check for new keywords appearing
- [ ] Look for any sudden drops in traffic
- [ ] Note any mobile usability issues

**Questions to Ask:**
- Are impressions trending up or down?
- Any new keywords in top 20?
- Any keywords that disappeared?
- Any mobile issues reported?

---

### Friday Afternoon Review

**Tasks (30 minutes):**
- [ ] Review week's top 10 keywords
- [ ] Calculate weekly average CTR
- [ ] Check for indexing errors
- [ ] Review mobile vs desktop performance
- [ ] Identify 1-2 optimization opportunities

**Questions to Ask:**
- What was the best-performing keyword this week?
- What keyword needs optimization?
- Are we on track for monthly goals?
- What should we optimize next week?

## Quarterly Review Process

### Every 3 Months: Comprehensive Strategy Review

**Step 1: Analyze Performance (1 hour)**
- Compare quarter to previous quarter
- Calculate growth rates
- Identify trends
- Celebrate wins

**Step 2: Review Keywords (1 hour)**
- Top 20 keywords
- New keywords appearing
- Keywords losing position
- Keywords to target next

**Step 3: Identify Opportunities (1 hour)**
- High impression, low CTR keywords
- Keywords ranking 11-20
- Featured snippet opportunities
- Content gaps

**Step 4: Plan Next Quarter (1 hour)**
- Content calendar (12 blog posts)
- Optimization priorities (top 10 keywords)
- Backlink strategy (50+ targets)
- Featured snippet targets (5+ snippets)

**Step 5: Report to Stakeholders (30 minutes)**
- Share performance metrics
- Highlight improvements
- Outline next steps
- Set expectations

## Integration with Google Analytics

### Linking GSC to Analytics

**Why Link?**
- See which keywords drive conversions
- Track user behavior after clicking
- Identify high-value keywords
- Optimize landing pages

**How to Link:**
1. Go to Google Analytics
2. Admin → Property Settings
3. Search Console Settings
4. Select your GSC property
5. Save

**What You'll See:**
- Keywords in Analytics
- Landing pages by keyword
- Conversion rate by keyword
- User behavior by keyword

## Troubleshooting Common Issues

### Issue: No Data in GSC

**Cause:** Website not verified or indexed  
**Solution:**
1. Verify website in GSC
2. Submit sitemap
3. Wait 2-4 weeks
4. Check robots.txt

### Issue: High Impressions, Low CTR

**Cause:** Poor meta description or title  
**Solution:**
1. Improve meta description
2. Add compelling title
3. Add rich snippets
4. Improve content

### Issue: Keyword Ranking Declining

**Cause:** Content outdated or competitors improving  
**Solution:**
1. Update content
2. Add more comprehensive info
3. Build backlinks
4. Improve internal linking
