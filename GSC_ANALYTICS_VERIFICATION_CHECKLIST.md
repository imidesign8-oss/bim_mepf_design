## 🛠️ TROUBLESHOOTING GUIDE

### Issue 1: No Data in GSC After 48 Hours

**Symptoms:**
- GSC shows "No data available"
- Sitemap status shows "Error"
- Coverage shows 0 pages indexed

**Solutions:**
1. **Verify DNS Record**
   - Go to your domain registrar
   - Check TXT record is correctly added
   - Verify there are no typos
   - Wait 24-48 hours for DNS propagation

2. **Check Robots.txt**
   - Visit https://imidesign.in/robots.txt
   - Verify it doesn't block crawling
   - Test in GSC "Robots.txt Tester"

3. **Resubmit Sitemap**
   - Go to GSC → Sitemaps
   - Delete current sitemap
   - Resubmit sitemap.xml
   - Wait for processing

4. **Contact Support**
   - If issues persist after 72 hours
   - Contact Google Search Console support

### Issue 2: Analytics Not Tracking

**Symptoms:**
- Google Analytics shows 0 sessions
- Real-time report is empty
- No data in Analytics dashboard

**Solutions:**
1. **Verify Tracking Code**
   - Open browser console (F12)
   - Search for "gtag" or "analytics"
   - Verify code is loaded
   - Check for JavaScript errors

2. **Check Analytics Settings**
   - Go to Google Analytics
   - Admin → Property Settings
   - Verify tracking is enabled
   - Check data retention settings

3. **Clear Browser Cache**
   - Clear browser cache and cookies
   - Visit website in incognito mode
   - Check if data appears in real-time report

4. **Wait 24 Hours**
   - Analytics takes up to 24 hours to process data
   - Check again after 24 hours

### Issue 3: Pages Not Indexed

**Symptoms:**
- GSC shows "Discovered but not indexed"
- Coverage report shows 0 indexed pages
- Pages not appearing in Google search

**Solutions:**
1. **Check for Noindex Tag**
   - View page source
   - Search for "noindex"
   - Should NOT have `<meta name="robots" content="noindex">`

2. **Check Page Quality**
   - Ensure pages have sufficient content
   - Add meta descriptions
   - Improve page titles
   - Add internal links

3. **Request Indexing**
   - In GSC, go to URL Inspection
   - Enter page URL
   - Click "Request Indexing"
   - Wait for processing

4. **Build Backlinks**
   - Get links from other websites
   - Improves page authority
   - Helps with indexing
